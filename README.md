# 🏰 Amunì Tourism - Caccamo Experience

Benvenuti nel cuore della Sicilia. **Amunì Tourism** è un'esperienza digitale interattiva dedicata alla scoperta del Castello di Caccamo e delle sue leggende, guidata dalla voce della **Castellana**.

## 🌟 Caratteristiche del Progetto
* **Avatar Interattivo**: Incontro con la Castellana di Caccamo, ricostruita digitalmente per narrare la storia del castello.
* **Atmosfera Immersiva**: Un layout moderno in stile "Glassmorphism" con uno sfondo dinamico che cattura i colori del tramonto siciliano.
* **AI-Powered Chat**: Un sistema di intelligenza artificiale (in fase di sviluppo) per dialogare con i custodi della storia.

## 🛠️ Tecnologie Utilizzate
* **HTML5 & CSS3**: Per la struttura e il design responsivo.
* **Python (Pillow)**: Utilizzato per l'elaborazione delle immagini e la gestione della trasparenza degli asset.
* **GitHub Pages**: Per l'hosting gratuito e la pubblicazione del sito live.

## 📂 Struttura del Repository
* `/asset`: Contiene le immagini elaborate (Castello, Castellana, Abito).
* `/css`: File di stile per il layout e gli effetti visivi.
* `/bot`: Logica Python per il "cervello" della Castellana (In corso).
